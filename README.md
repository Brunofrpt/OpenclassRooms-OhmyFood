# OhMyFood – Projet OpenClassrooms

Projet réalisé dans le cadre de la formation **Intégrateur Web – OpenClassrooms**.

## Objectif
Intégration du site OhMyFood en HTML et CSS (Sass), en mobile first, avec animations.

## Technologies utilisées
- HTML5
- CSS3 / Sass (architecture 7-1)
- Git & GitHub
- visual studio code

## Lancer le projet en local
- Ouvrir `OhMyFood/index.html` dans un navigateur

### Compilation Sass
-depuis OhMyFood lancer :

npx sass --watch SASS/style.scss style.css
